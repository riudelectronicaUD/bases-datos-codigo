# Análisis de Deserción Estudiantil
Este proyecto presenta un análisis de la deserción estudiantil a partir de la integración de bases de datos de estudiantes admitidos y matriculados desde 2017-1.

## Contenido

- Construcción de base de datos consolidada en Python
- Análisis de cohortes
- Cálculo de deserción total y por periodo
- Visualización en Power BI

## Metodología

Se integraron dos bases de datos:
- Estudiantes admitidos
- Estudiantes matriculados
